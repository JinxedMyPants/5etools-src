    Nutzen Sie Discord?

    Treten Sie dem Fan-Server für Ankündigungen, Updates und Feature-Abstimmungen bei.

    Nutzt du Foundry?

    Nein? Nun, das sollten Sie! Foundry ist ein modernes, besseres Ersatzprodukt für Roll20 und legt Wert auf Modding-Unterstützung. Es ist die bevorzugte Plattform für VTT-Integrationen von 5etools.

    Um das Plutonium-Modul zu installieren und alle 5etools-Inhalte in Ihrem Spiel zu verwenden, fügen Sie die Manifest-URL[aktuell/v13/v12] in den Modul-Installer Ihres Foundry-Servers ein. Siehe die Plutonium-Features-Seite für eine Übersicht über die bereitgestellte Funktionalität.

    Sie können auch die Rivet-Browsererweiterung (Chrome/Firefox) installieren, um Inhalte mit einem Klick von der Seite zu importieren.

    Nutzen Sie Roll20?

    Installieren Sie Tampermonkey (Chrome/Firefox), und das betteR20-Skript[2014/2024/Jumpgate 2014 (experimentell)/Jumpgate 2024 (experimentell)], die alle das D&D 5E by Roll20-Sheet benötigen. Es sollte immer nur ein Skript gleichzeitig aktiv sein.

    Siehe die Wiki für eine Installationsanleitung und Informationen zur Verwendung des Skripts.

    FAQ

    Gibt es ein Wiki? Hier. Beiträge sind willkommen.

    Ist es kostenlos? Ja! Sie können sogar den Quellcode herunterladen[aktuell/2014]. Siehe die README oder das Wiki für Hilfe.

    Kann ich beitragen? Der Quellcode ist auf GitHub verfügbar[aktuell/2014]. Siehe die Contributing-Anleitung.

    Browser- und Geräteunterstützung? Die neuesten Desktop-Versionen von Chrome und Firefox.

    Wo sind meine Daten? Es gibt kein Kontosystem. Alles wird in Cookies gespeichert. Wenn Sie diese löschen oder Ihr Inkognito-Fenster schließen, sind die Daten verloren. Sie können Ihre Daten über das Einstellungsmenü herunterladen, um sie zu sichern.

    Andere Versionen? Eine „2014“-Version ist verfügbar sowie ein Archiv der Releases.

    Noch etwas? Halten Sie SHIFT beim Überfahren gedrückt, um ein Fenster zu fixieren. Alles hat Tooltips. Sie werden für immer Spielleiter sein.

    Self-Hosting

    Sie können eine Kopie von 5etools auf allem hosten, was einen einfachen Webserver ausführen kann — Beispiele sind Ihr Computer, Ihr Telefon, ein Raspberry Pi oder eine kostenlose EC2/Compute Engine-Instanz. Laden Sie die Dateien und Bilder herunter, kopieren Sie sie in das Stammverzeichnis Ihres Servers und genießen Sie!
